# NATURIO – Premium Organic Grocery Delivery App UI/UX

## Overview

NATURIO is a premium organic grocery delivery mobile application designed to provide users with a modern, clean, and healthy shopping experience.

The project focuses on creating a high-fidelity UI design that communicates freshness, trust, sustainability, and premium quality through visual design principles.

This project was developed as part of the UI/UX Industrial Training Program.

---

## Project Objective

The objective of this project is to design a modern organic grocery delivery application that provides:

- Fresh and healthy shopping experience
- Easy product browsing
- Attractive product presentation
- Premium brand identity
- User-friendly interface
- Clean and modern visual design

---

## Design Theme

NATURIO follows a luxury organic design approach inspired by modern grocery and lifestyle applications.

### Design Values

- Freshness

- Trust

## Screens Designed

### 1. Style Tile

The Style Tile defines the visual identity of the application including:

- Color palette
- Typography
- Components
- Icons
- Buttons
- Spacing system
- Image style

---

### 2. Splash Screen

The splash screen introduces the brand identity of NATURIO.

Features:
- Brand logo
- Tagline
- Organic product imagery
- Call-to-action button

---

### 3. Product List Screen

The home screen allows users to browse products.

Features:
- Search bar
- Categories section
- Promotional banner
- Product cards
- Bottom navigation

---

### 4. Product Details Screen

The product details screen provides detailed information about the selected product.

Features:
- Product image
- Price
- Ratings
- Description
- Quantity selector
- Add to Cart button
